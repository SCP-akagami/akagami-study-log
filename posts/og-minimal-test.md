# Minimal OG Image Test

最小限のSVGファイルを使用したOG画像テストです。

![OG Test Image](/images/og-test-minimal.svg)

## テスト内容

- 最小限のSVG要素のみを使用
- 複雑なフォントやグラデーションを使用しない
- Discord対応を確認

## 使用しているSVG要素

- `<rect>` - 背景と白いボックス
- `<text>` - シンプルなテキスト
- 標準的なフォント（Arial）のみ

このテストでDiscordでの表示が改善されるかを確認します。
